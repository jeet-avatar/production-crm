    ```
    const STAGE_MAP: Record<string, string> = {
      'prospecting': 'PROSPECTING',

      'proposal': 'PROPOSAL',
      'proposal/price quote': 'PROPOSAL',

    };
    ```

    Processing loop logic:
    1. Validate: skip if title is empty string or missing. Skip if parseFloat(value) is NaN. Push error entry for each skipped row with reason.
    2. Resolve contactId: if contactEmail provided, query prisma.contact.findFirst({ where: { email: contactEmail } }). Use id if found, null otherwise (do NOT fail the row — just import without contact link).
    3. Resolve companyId: if companyName provided, query prisma.company.findFirst({ where: { name: { contains: companyName, mode: 'insensitive' } } }). Use id if found, null otherwise.
    4. Normalize stage: apply STAGE_MAP lookup. If not in map, default to 'PROSPECTING'.
    5. Call prisma.deal.create({ data: { title, value: parsedValue, stage: normalizedStage, probability: probability ?? 10, expectedCloseDate: expectedCloseDate ? new Date(expectedCloseDate) : null, contactId, companyId, description, userId: req.user.id } }).
    6. Wrap each create in try/catch — catch pushes to errors array with deal title and error message.

    Response shape:
    ```
    { imported: number, failed: number, errors: Array<{ row: number, title: string, reason: string }> }
    ```
