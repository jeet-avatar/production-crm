```prisma
model Quote {
  id         String      @id @default(cuid())
  dealId     String
  userId     String
  status     QuoteStatus @default(DRAFT)
  title      String
  lineItems  Json
  subtotal   Decimal     @db.Decimal(12, 2)
  tax        Decimal?    @db.Decimal(12, 2)
  total      Decimal     @db.Decimal(12, 2)
  notes      String?
  validUntil DateTime?
  sentAt     DateTime?
  acceptedAt DateTime?
  createdAt  DateTime    @default(now())
  updatedAt  DateTime    @updatedAt

  deal      Deal       @relation(fields: [dealId], references: [id], onDelete: Cascade)
  user      User       @relation(fields: [userId], references: [id], onDelete: Cascade)
  contracts Contract[]

  @@index([dealId])
  @@index([userId])
  @@index([status])
  @@map("quotes")
}
```

3. Add the `Contract` model:

```prisma
model Contract {
  id        String         @id @default(cuid())
  dealId    String
  quoteId   String?
  userId    String
  status    ContractStatus @default(DRAFT)
  title     String
  content   String         @db.Text
  variables Json?
  signedAt  DateTime?
  signedBy  String?
  createdAt DateTime       @default(now())
  updatedAt DateTime       @updatedAt

  deal  Deal    @relation(fields: [dealId], references: [id], onDelete: Cascade)
  quote Quote?  @relation(fields: [quoteId], references: [id], onDelete: SetNull)
  user  User    @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@index([dealId])
  @@index([userId])
  @@index([status])
  @@map("contracts")
}
```

**Create `backend/src/routes/quotes.ts`:**

Copy the structural pattern from `backend/src/routes/deals.ts`. The router must:
- Import: `express`, `{ PrismaClient }` from `@prisma/client`, `{ authenticate }` from `../middleware/auth`
- `router.use(authenticate)` at top (all quote endpoints require auth)
- All Decimal fields (`subtotal`, `tax`, `total`) must be serialized with `Number()` before `res.json()` — never return raw Prisma Decimal objects
- Ownership check on every mutation: `where: { id, userId: req.user?.id }`

Implement these endpoints:

`GET /` — List quotes filtered by `?dealId=`:
```typescript
const quotes = await prisma.quote.findMany({
  where: { dealId: req.query.dealId as string, userId: req.user!.id },
  orderBy: { createdAt: 'desc' },
});
res.json(quotes.map(q => ({ ...q, subtotal: Number(q.subtotal), tax: q.tax ? Number(q.tax) : null, total: Number(q.total) })));
```

`POST /` — Create quote. Required body fields: `dealId`, `title`, `lineItems` (array), `subtotal`, `tax` (optional), `total`. Set `userId: req.user!.id`. Validate deal exists and belongs to user before creating.

`PUT /:id` — Update quote. Only update fields provided in body. Ownership check via `where: { id, userId: req.user!.id }`. Serialize Decimals on response.

`PATCH /:id/status` — Update only the `status` field. Body: `{ status: QuoteStatus }`. Ownership check required. Sets `sentAt: new Date()` when status becomes SENT, `acceptedAt: new Date()` when status becomes ACCEPTED.

`DELETE /:id` — Hard delete. Ownership check: `where: { id, userId: req.user!.id }`.

`GET /` — List contracts filtered by `?dealId=`. Include `where: { dealId, userId: req.user!.id }`, order by `createdAt: 'desc'`.

`PUT /:id` — Update contract (title, content, variables). Ownership check required.
