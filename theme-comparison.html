Trade-offs

Option A: Orange / Rose

  • Already live on production (seconf branch)
  • Has newer features: AIChat, Video Campaigns, Logo component, branding config, Super Admin
  • Warm, energetic — feels approachable for SMB CRM
  • Light theme = better readability in bright environments
  • Generic SaaS look — similar to many CRMs
  • Orange/rose can feel less "premium" or "enterprise"
  • No glassmorphism or depth effects

Option B: Indigo Noir

  • Premium, high-end look (Linear, Raycast, Arc aesthetic)
  • Glassmorphism, ambient blobs, spring animations
  • Dark mode = modern, reduces eye strain
  • Stands out from competitor CRMs visually
  • Needs rebase onto seconf — merge conflicts (~15 files)
  • Missing newer seconf features (AIChat, Video, branding config)
  • Dark themes can reduce readability for data-heavy tables
  • Can add the seconf features after rebase (~2-3 hours)

Decision

Option A is safer — it's already live with all features.
Option B is bolder — premium feel but needs rebase work.

A third option: keep Option A's features and apply Indigo Noir on top (rebase). This gives you both the new features AND the premium dark look.
